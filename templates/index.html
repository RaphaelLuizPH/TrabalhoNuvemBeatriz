Introdução ao Google Cloud


O Google Cloud é uma plataforma de computação em nuvem que permite construir, implantar e escalar aplicativos, sites e serviços na mesma infraestrutura do Google dados, multi-cloud híbrida, IA e ML1.

O Google Cloud oferece mais de 20 produtos gratuitos e novos clientes recebem $300 em créditos gratuitos na inscrição

Recursos do Google Cloud


Recursos do Google Cloud IA:

Aprendizado de máquina: O Google Cloud permite construir e dimensionar aplicativos de IA com o Vertex AI e o Gemini Code Assist.

Computação: O Google Cloud oferece uma variedade de opções de computação para atender às suas necessidades, desde VMs até contêineres e funções sem servidor.

Armazenamento: O Google Cloud oferece soluções de armazenamento duráveis, seguras e escalonáveis para seus dados estruturados e não estruturados

Introdução ao Cloud Run


O Cloud Run é uma plataforma de computação gerenciada que permite executar contêineres diretamente na infraestrutura escalável do Google.

Você pode implantar código escrito em qualquer linguagem no Cloud Run se puder construir uma imagem de contêiner a partir dele.

O Cloud Run foi projetado para funcionar bem com outros serviços do Google Cloud, permitindo a construção de aplicativos completos.

4 / 4

Recursos do Cloud Run


Serviços e trabalhos: No Cloud Run, seu código pode ser executado continuamente como um serviço ou como um trabalho.

Escalabilidade rápida baseada em solicitações: O Cloud Run é construído para escalar rapidamente para lidar com todas as solicitações recebidas ou para lidar com o aumento da utilização da CPU fora das solicitações se a alocação de CPU estiver definida para sempre ligada.

Gerenciamento de tráfego integrado: Cada implantação cria uma nova revisão imutável. Você pode rotear o tráfego de entrada para a revisão mais recente, voltar para uma revisão anterior ou dividir o tráfego para várias revisões ao mesmo tempo, para realizar uma implantação gradual.
